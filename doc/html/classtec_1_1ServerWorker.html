Detailed Description

template<typename TParams, typename TServer>
class tec::ServerWorker< TParams, TServer >

A worker that owns and manages a server instance.

Extends the Worker class to start and shut down a server instance, handling server lifecycle within a dedicated thread. It uses signals and status objects to manage server startup and shutdown with timeout support.
